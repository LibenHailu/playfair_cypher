The Playfair Cipher Encryption Algorithm

Input a keyword below to seed the key table. All keywords are converted to uppercase and should contain only alphabetic characters. Any non-alphabetic characters and duplicate characters are automatically stripped. The key table is filled in with the rest of the unused alphabet.

Note: The letter J is replaced with the letter I in all digraphs. A digraph is what we convert your plain text message to. It is a pairing of letters, with X used to break up duplicate letters and to pad uneven pairs. The spacing is not retained once the plain text is converted to a digraph.
